== Installation ==

This section describes how to install the plugin and make it working.

1. Upload `customize-login` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings/customize login to edit your settings.
4. Design away.
5. checkbox check for "enable" custom login screen.

For more details please visit our website
